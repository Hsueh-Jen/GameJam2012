# Jizo Runner (太陽升起之前)

這是一款復刻 2012 年福島 Game Jam 的橫向卷軸小遊戲。
玩家扮演一名深夜的使者，在太陽升起之前，收集各種道具並將其交給路邊的地藏小菩薩，幫助祂們化為人形並一同迎接黎明。

當時做出來真的很不好玩，所以想要復刻一下當年的糞作，試著查出不好玩的原因。
復刻版本使用 Google Antigravity 實作，大概花了4~5個小時完成。(製作時間：2026/3/15 10:30pm ~ 2026/3/16 2:30am)

## 遊戲核心 (Core Mechanics)

1.  **遊戲引擎**：使用 HTML5 `<canvas>` 與原生 JavaScript (Vanilla JS)。
2.  **動態環境**：
    -   **自動捲軸**：攝影機會隨著時間持續向右推進，挑戰玩家的反應與移動。
    -   **視差背景**：擁有遠景高山與中景森林的視差捲軸效果，增加畫面深度。
    -   **日出計時**：畫面上方有日出進度條，背景顏色會隨著時間從漆黑深夜漸漸轉為橙紅晨曦，當太陽升起時遊戲結束。
3.  **道具與推車系統**：
    -   主角身後拖著一台**木製推車**，顯示當前持有的道具。
    -   主角手中一次只能持有**一個**道具。
4.  **地藏與跟隨者**：
    -   地藏小菩薩頭頂會有需求泡泡，顯示祂想要的物品（💣、🍎、🎈）。
    -   當主角持有正確物品並靠近時，物品會**自動交付**。
    -   成功交付後，地藏會轉化為**天使/跟隨者**，依照歷史軌跡加入主角的隊伍。
5.  **音效與氛圍**：
    -   包含背景音樂 (BGM) 與程序化生成的音效（拾取、交付、煙火發射與爆炸）。
    -   遊戲結束時有精彩的煙火慶典畫面。

## 道具說明 (Items)

-   💣 **炸彈**：位於地面的道具。主角必須蹲下（方向鍵向下）才能撿起。
-   🍎 **蘋果**：長在樹上的道具。主角跳躍並觸碰到即可自動收集。
-   🎈 **氣球**：在空中上下飄浮的道具。主角跳躍並觸碰到即可自動收集。

## 控制說明 (Controls)

-   **左右移動**：使用 `←` `→` 方向鍵。放開時主角仍會隨攝影機速度自動前行。
-   **跳躍**：按下 `↑` 或 `W` 鍵。
-   **撿拾地面道具**：按下 `↓` 或 `S` 鍵（主角會做出蹲下撿拾動作）。
-   **交付道具**：持有正確道具時**靠近**地藏小菩薩，系統會自動處理交付。
-   **開始/重新遊戲**：在選單畫面按 `SPACE` 或 `ENTER` 鍵。

## 技術細節 (Technical Details)

-   **座標系統**：使用 World X 與 Camera X 分離的邏輯。
-   **碰撞偵測**：基於 AABB 與距離判定實現互動邏輯。
-   **歷史軌跡追蹤**：跟隨者系統透過記錄玩家的座標歷史，實現流暢的蛇行移動。
-   **程序化生成**：背景景觀（山脈、樹木）與道具地藏皆為動態生成與回收。

## 如何執行 (How to Run)

只需在瀏覽器中開啟 `index.html` 即可開始遊戲。建議使用 Chrome 或 Edge 瀏覽器以獲得最佳音效體驗。
